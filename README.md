# worktype_salary_distribution_boxplot
This project is an interactive Tableau dashboard that analyzes salary distribution for internship roles using a box-and-whisker plot. It focuses on understanding how salaries vary under different filtered conditions.

 
 Project Overview

This visualization explores internship salary patterns by applying multiple filters such as work type, company size, experience, and job constraints. The goal is to analyze how salaries are distributed among intern roles.


 Objective

To analyze internship salary distribution and identify patterns using a box-and-whisker plot with strict filtering conditions.


 Dataset Filters Used

Work Type = Intern
Latitude < 10
Company Size < 50000
Salary > 8000
Job Title: Single word & < 10 characters
Experience: Even numbers only
Posting Date: 2021–2023
Contact Person contains letter "e"
Time filter: Tableau Desktop displays results in IST (3 PM – 5 PM), while Tableau Public follows UTC time zone behavior, affecting visibility timing
 
 Visualization

Type: Box-and-Whisker Plot
Columns: Work Type
Rows: Salary (AVG/Distribution)
Purpose: Salary distribution analysis for internship roles
🛠 Tools Used
Tableau Public

Data Visualization Techniques
Calculated Fields & Filters


 Key Insights

Internship salaries vary significantly across filtered conditions
Salary distribution shows outliers and median trends clearly
Work type filtering helps isolate internship-specific insights
Strict conditions improve data quality and accuracy
